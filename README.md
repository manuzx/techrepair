O projeto consiste na criação de uma landing page(Página única) para a assistência técnica de computadores TechRepair Fortaleza, conforme solicitado pelo cliente, Sr. Roberto. O objetivo é entregar um site simples, moderno e de alta manutenibilidade, seguindo rigorosamente as metodologias de gestão de qualidade

Tecnologias utilizadas:

- HTML5:Estrutura e semântica
- CSS3: Estilização e resposividade
- Font awesome: Ícone das redes sociais.
- Live server: Criação do servidor local para a pré-visualização do site

Metodologia de gestão(PDCA)

A. Planejamento (PLAN)

-Ferramenta: Quadro trello(kanban)
-Colunas: Backlog, A fazer, Em teste, Concluido.

B. Execução(DO)

O código foi escrito com foco na manutenção preventiva 


Requisito:

1. Semântica: Tags separadas de forma a facilitar uma futura manutenção

2. Comentários: Todo o código está comentado, facilitando a leitura e reduzindo a dívida técnica pelo fato do codigo ser auto-explicativo
